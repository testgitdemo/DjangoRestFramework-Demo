My Site Django App.

My Site is web application develop in django framework.In this site manage different modules like Student Detail,University Detail and Employees registration,login,forgot password etc. Here also manage admin works like manage Student detail and it's contact detail and Employee's Department Detail.

Student Details

Student detail define list of student manage by admin.We can show list of students and update each students Contact detail like Contact Number.

Employee Detail

Here Employee can register with it's department that manage by admin. The register employees can login and manage self profile detail.If any employees lost password just give register mobile number and get the password vai register email.

University Detail

The University Detail defiles university basic information.We can insert new unversity and each university contain it's Courses Detail we can show,edit and delete Course as well as university details.Here Perform all CRUD Operation on Django

Admin Detail

Here admin manage My Site.Admin can add new student and student contact detail for Student Detail, Admin can add Department for Employee Detail and many more.
